## 核心结论

1. 罗福莉认为 2026 年的关键变化是从 Pre-train 主导的 Chat 时代，进入 Post-train 主导的 Agent 时代。
2. OpenClaw 被她视为 Agent 框架层面的重大事件，而不是 Claude Code 外面套 UI 的产品包装。
3. Agent 范式会改变训练目标：模型不仅要会回答，还要能在工具、环境、长链路任务和反馈中自我推进。
4. 国内顶尖团队在 Pre-train 上的差距已经显著缩小，下一阶段竞争焦点转向 Agent post-training 和 RL scaling。
5. 用卡比例会发生结构变化：Chat 时代 Pre-train 占绝对大头，Agent 时代 Pre-train 与 Post-train 可能接近 1:1。
6. 1T 模型是下一阶段入场券，但真正的战略问题是 1T 之后 scaling 什么：参数、环境、推理、post-train、数据还是芯片。
7. RL/Agent Infra 与 Pre-train Infra 的工程气质不同，前者必须容忍中断、异构资源、超时、工具失败和复杂验证。
8. 组织上，罗福莉强调 diversity 和环境：经验会快速折旧，把人放进正确反馈环境比找“有经验的人”更关键。

- 02:38:27 每天在否认昨天的自己：这里讲个人和团队认知迭代。过去稳定的经验在新范式里会不断被推翻，团队需要允许昨天的判断被今天的实验否定；这种快速自我修正比维护既有路线更重要。
- 02:47:49 过去3年的AI进化史：她回顾 2023-2026 年的主线变化：从大模型预训练追赶，到 Chat 产品爆发，再到工具调用、Agent 框架、Post-train 和 RL 环境成为新焦点。这个历史回顾用于解释为什么今天的组织和算力配置必须重新排序。
- 03:05:09 当下共识与竞争：罗福莉认为 Anthropic/Claude Code 代表的路线已经让行业形成新共识：模型能力、框架、工具、推理系统和环境反馈要一起看。接下来竞争会考验团队是否能在很短周期内吸收共识并转成工程结果。
- 03:19:00 环境比经验更重要：结尾回到人才和组织判断。新范式下，“有经验”不如“处在高反馈环境里”重要；团队要把人放到能快速接触真实任务、失败样本、用户反馈和训练信号的位置，才能持续更新能力。

## 技术判断

- Agent 框架会显著抬高同一基座模型的可用上限，框架本身成为技术变量。
- Post-train 不再只是 SFT/RLHF 的常规尾部工序，而会成为模型公司主战场。
- Agentic 能力需要 RL scaling，因为真实任务有环境、工具、验证和多步反馈。
- Pre-train Infra 追求稳定吞吐和无 spike；RL/Agent Infra 要处理异构、超时、重试、评测、工具失败和推理/训练不一致。
- 1T 参数规模只能提供入场资格，决定领先的可能是后训练、环境构建、推理系统和芯片/算力配置。
- OpenClaw/Claude Code 类系统显示，能力释放不只来自模型权重，也来自外部 harness。
- 另类架构的价值不只在模型结构本身，也在外部工具、记忆、环境和多模型协作如何降低长任务的不确定性。

## 商业与组织判断

- 小米作为终端公司拥有手机、汽车、IoT 和人车家生态，长期可能给 Agent 提供丰富环境和反馈。
- 资源配置会从“Pre-train 压倒一切”转向“研究、Pre-train、Post-train 更均衡”。
- 团队要快速拥抱新范式，不能把 Pre-train 时代的指标、流程和 infra 原封不动搬到 Agent 时代。
- diversity 是组织能力：算法、工程、产品、infra、评测和环境构建必须混合。
- “环境比经验更重要”对应的是组织设计：让成员进入真实任务和高密度反馈，而不是只按过去履历分配决策权。

## 风险与争议点

- OpenClaw 是否会长期决定训练范式，仍需观察；它可能是阶段性 framework shock，也可能成为新标准。
- Agent post-training 的 scaling law 还不清晰，投入是否能稳定兑现尚未被长期验证。
- 第三方 harness 的 token 消耗和成本可能很高，商业化压力会倒逼效率优化。
- 1T 之后到底 scaling 参数、环境、推理还是芯片，仍是开放问题。
- 组织快速转向会带来焦虑和资源冲突，尤其是 Pre-train 与 Post-train 团队之间的优先级重排。
