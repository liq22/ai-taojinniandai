## 核心结论

1. 这期不是单纯商业访谈，而是一条研究者思想形成史：谢赛宁从视觉、表征学习、学术流浪、人际影响，走向世界模型和 AMI Labs。
2. 他对硅谷 LLM 路线提出强烈批评，认为“Silicon Valley is very LLM-pilled”，语言模型并不是通向智能的唯一甚至最可靠路径。
3. 视觉不是附属模态，而是世界理解的主入口；语言如果过度主导，可能污染视觉和世界表征。
4. 他把 representation learning 放在核心位置：智能要形成对世界的内部表征，而不是只在语言中复述人类文本。
5. 世界模型的关键是从真实世界、视频、行动和反馈中学习；只“下载互联网”或“下载人类”都不够。
6. AMI Labs 的创业是对 OpenAI 式路径的反方向实验：小而高密度的研究组织、Yann LeCun 路线、长期世界模型目标。
7. 访谈中的人物线很重要：何恺明、杨立昆、李飞飞、Ilya 等人不是八卦，而是谢赛宁研究 taste 和价值观的来源。
8. 最大风险是世界模型路线更慢、更难验证，也更依赖长期资本和研究耐心；但它可能触及 LLM 路线难以解决的物理世界智能问题。

## 技术判断

- LLM 并不是 Bitter Lesson 的简单胜利样本，因为它依赖高度人类加工的文本和偏好，而不是直接从环境中学习。
- 视觉和视频数据携带空间、运动、因果和物理规律，是世界模型不可绕开的核心输入。
- 语言可能污染视觉：如果视觉表征过度被语言标签牵引，模型会失去对原始世界结构的敏感性。
- Diffusion Transformers / Sora 一类工作说明生成模型正从图像生成走向时空建模，但它们距离完整世界模型仍有距离。
- 世界模型需要预测、行动、反馈和自校正，不只是生成逼真视频。
- AMI Labs 的路线更接近表征学习、真实世界学习和长期智能系统，而非继续堆叠聊天能力。

## 商业与组织判断

- AMI Labs 属于 Neo Labs 现象：顶级研究者、小团队、大额融资、长期研究导向。
- 小团队能保持研究 taste 和方向一致，但也必须承受资本对进度、叙事和结果的压力。
- “反 OpenAI”更准确地说是反单一路径霸权：不是拒绝大模型，而是拒绝把 LLM 当成唯一答案。
- 世界模型商业化短期不如 Agent 应用清晰，长期可能落到机器人、自动驾驶、仿真、科学发现和具身智能。

## 风险与争议点

- LLM 路线仍在快速进步，过早判断其上限可能有风险。
- 世界模型的评测、数据、算力和产品化路径都还不清晰。
- AMI Labs 融资规模巨大，若短期没有可展示结果，容易被外界按 LLM 公司标准误判。
- 对语言的批评很有启发，但语言仍是人类知识和协作的主要载体，不能简单视为负资产。
